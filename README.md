## Features
- 生年月日と気分から今日の運勢を占う
- 星座判定とその特徴・アドバイス表示
- タロットカード（大アルカナ・小アルカナ78枚）の抽選と意味
- 五行によるラッキーカラーと今日のミッション生成
- 運気スコア算出
- 過去の占い履歴表示
- SNSシェア用テキスト生成

## Requirements
- Python 3.10以上
- Streamlit==1.30.0

## Usage
- 生年月日を入力
- 気分を選択
- 「運勢を占う」ボタンをクリック
- 星座、タロット、五行、今日の運勢、ミッションが表示されます
- SNSシェア用テキストも確認可能
